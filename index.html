PIRB by Pythonz

A modularized IRC-bot written in Python

We try to create an eggdrop alternative (of cause we know, no bot will be better as an eggdrop). It won't need many ressources, stores it data in sqlite3 databases and can load/reload/unload modules and the src on the fly (of course, you shouldnt play in src directory until you know what you're doing)!

Dependencies

Python... Nothing else. If you want to create own modules. Maybe a bit brain.

Install

Download Type: './run.py configure' (this will setup the config easily)! Run './run.py' or the restarter './start'
